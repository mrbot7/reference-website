INTRO

Random numbers are useful in software development for many reasons-game mechanics, generating usernames/passwords, anytime you need something seemingly unpredictable
Pseudo-random numbers- computers are just big calculators solving formulas and can't create truly random numbers, but instead use an algorithm that generates numbers that seem random

Notes

Random numbers in JS

Javascript provides a numbers of functions that are useful for generating random numbers

  • Math.random() is a function that returns a pseudo-random number between 0 and 1, not including 1
  • Math.floor() is a function that takes a decimal number and rounds it down - cops off the decimal part

Notes

Random number calculations

To produce a random integer from a min value to a max value (including the min and max) use this calculation
Math.floor(Math.random() * (max + 1)) + min

Example

Generate a random number from 1 to 10 and store it in a variable
var randNum = Math.floor(Math.random() * 11) + 1;
